# digit-recognition-svm
# Digit Recognition using SVM

This project implements a Digit Recognition System using Support Vector Machine (SVM).

## Dataset
Scikit-learn digits dataset (8x8 images of digits 0–9)

## Algorithm
Support Vector Machine (SVM) with RBF kernel

## Steps
- Load dataset
- Train-test split
- Train SVM model
- Predict digits
- Evaluate performance

## Accuracy
~95% using SVM (RBF Kernel)

## Future Scope
Can be improved using Deep Learning (CNN)

## How to Run
pip install -r requirements.txt  
python digit_svm.py
